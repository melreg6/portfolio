Overview of Project

A signal processing machine with UART and FPGA is used to darken, brighten, and invert images. MATLAB interfaces with the system, enabling efficient real-time manipulation of images through hardware-based processing.

System Description

In MATLAB, we read and resize the image, convert it to 8-bit RGB values, and send data byte-by-byte over UART. The receiver and transmitter each use FSMs: one to receive bytes into a shift register, the other to send them out. This processed data is displayed alongside the original image.

Challenges

A major challenge was enabling color filtering through UART. Due to continuous transmission and MATLAB's live reading, attempts to store data in buffers or FIFOs resulted in incorrect images (black outputs). Future iterations will focus on solving this issue by improving data synchronization and storage logic.
